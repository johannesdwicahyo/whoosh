# Whoosh v0.4.0: Real-World Readiness Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add features needed for real deployment: graceful shutdown, request ID propagation, request-scoped logging, health check endpoint, and config validation.

**Architecture:** Graceful shutdown hooks into SIGTERM/SIGINT. Request ID flows through env hash (`whoosh.request_id`), set by RequestLogger middleware, accessible everywhere. Logger gains a `with_context` method for request-scoped fields. Health check is a built-in route with configurable probes.

**Tech Stack:** Ruby 3.4+, RSpec, rack-test.

**Depends on:** v0.3.0 complete (347 tests passing).

```ruby
# lib/whoosh/middleware/request_logger.rb
# frozen_string_literal: true

require "securerandom"

module Whoosh
  module Middleware
    class RequestLogger
      def initialize(app, logger:)
        @app = app
        @logger = logger
      end

      def call(env)
        # Set request ID early so all downstream code can access it
        request_id = env["HTTP_X_REQUEST_ID"] || SecureRandom.uuid
        env["whoosh.request_id"] = request_id

        start_time = Process.clock_gettime(Process::CLOCK_MONOTONIC)

        status, headers, body = @app.call(env)

        duration_ms = ((Process.clock_gettime(Process::CLOCK_MONOTONIC) - start_time) * 1000).round(2)

        # Echo request ID in response
        headers["x-request-id"] = request_id

        @logger.info("request_complete",
          method: env["REQUEST_METHOD"],
          path: env["PATH_INFO"],
          status: status,
          duration_ms: duration_ms,
          request_id: request_id
        )

        [status, headers, body]
      end
    end
  end
end
```

- [ ] **Step 2: Write implementation**

```ruby
# lib/whoosh/shutdown.rb
# frozen_string_literal: true

module Whoosh
  class Shutdown
    def initialize(logger: nil)
      @hooks = []
      @logger = logger
      @executed = false
    end

    def register(&block)
      @hooks << block
    end

    def execute!
      return if @executed
      @executed = true

      @hooks.reverse_each do |hook|
        hook.call
      rescue => e
        @logger&.error("shutdown_hook_error", message: e.message)
      end
    end

    def install_signal_handlers!
      trap("TERM") { execute! }
      trap("INT") { execute! }
    end
  end
end
```

```ruby
RSpec.describe "Health check" do
  include Rack::Test::Methods

  def app = application.to_rack

  describe "default /healthz" do

      application.health_check
      application.get("/test") { { ok: true } }

    it "returns 200 with status ok" do
      get "/healthz"
      expect(last_response.status).to eq(200)
      body = JSON.parse(last_response.body)

      get "/healthz"
      body = JSON.parse(last_response.body)
      expect(body["version"]).to eq(Whoosh::VERSION)
    end
  end

  describe "with custom probes" do
    it "reports probe results" do
      application.health_check do
        probe(:database) { true }
        probe(:cache) { true }
      end
      application.get("/x") { {} }

      get "/healthz"
      body = JSON.parse(last_response.body)
      expect(body["checks"]["database"]).to eq("ok")
      expect(body["checks"]["cache"]).to eq("ok")

    it "returns 503 if any probe fails" do

        probe(:database) { true }
        probe(:cache) { raise "connection refused" }
      end
      application.get("/x") { {} }

      get "/healthz"
      expect(last_response.status).to eq(503)
      body = JSON.parse(last_response.body)
      expect(body["status"]).to eq("degraded")
      expect(body["checks"]["cache"]).to include("fail")
    end
  end
end
```

- [ ] **Step 2: Write implementation — add to App**

Read `lib/whoosh/app.rb`. Add public method:

```ruby
    # --- Health check ---

    def health_check(path: "/healthz", &block)
      probes = {}
      if block
        builder = HealthCheckBuilder.new
        builder.instance_eval(&block)
        probes = builder.probes
      end

      get path do
        checks = {}
        all_ok = true

        probes.each do |name, probe_block|
          begin
            probe_block.call
            checks[name.to_s] = "ok"
          rescue => e
            checks[name.to_s] = "fail: #{e.message}"
            all_ok = false
          end
        end

        status = all_ok ? "ok" : "degraded"
        result = { status: status, version: Whoosh::VERSION }
        result[:checks] = checks unless checks.empty?

        if all_ok
          result
        else
          [503, { "content-type" => "application/json" }, [Serialization::Json.encode(result)]]
        end
      end
    end
```

Add builder class inside App:

```ruby
    class HealthCheckBuilder
      attr_reader :probes

      def initialize
        @probes = {}
      end

      def probe(name, &block)
        @probes[name] = block
      end
    end
```
