# Whoosh v0.3.0: Performance Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Achieve 40-60K req/s on simple JSON endpoints. Outperform FastAPI (~30K), approach Fastify (~50-80K). Implement layers A (Oj, YJIT) and B (allocations, compiled middleware, router optimization).

**Architecture:** Oj is an optional dependency auto-detected at boot. YJIT auto-enabled. Middleware compiled into single proc. Router uses flat hash for static routes. Response headers pre-frozen. Benchmark suite validates gains at each layer.

**Tech Stack:** Ruby 3.4+, RSpec, benchmark-ips. Optional: oj gem.

**Depends on:** v0.2.0 complete (340 tests passing).

```ruby
# lib/whoosh/serialization/json.rb
# frozen_string_literal: true

require "json"
require "bigdecimal"

module Whoosh
  module Serialization
    class Json
      @engine = nil
      @oj_available = nil

      class << self
        attr_reader :engine

        def detect_engine!
          @oj_available = begin
            require "oj"
            true
          rescue LoadError
            false
          end
          @engine = @oj_available ? :oj : :json
        end

        def use_engine(name)
          @engine = name.to_sym
        end

        def encode(data)
          detect_engine! unless @engine
          prepared = prepare(data)

          if @engine == :oj
            Oj.dump(prepared, mode: :compat)
          else
            JSON.generate(prepared)
          end
        end

        def decode(raw)
          return nil if raw.nil? || raw.empty?
          detect_engine! unless @engine

          if @engine == :oj
            Oj.load(raw, mode: :compat)
          else
            JSON.parse(raw)
          end
        end

        def content_type
          "application/json"
        end

        def prepare(obj)
          case obj
          when Hash
            obj.transform_values { |v| prepare(v) }
          when Array
            obj.map { |v| prepare(v) }
          when Time, DateTime
            obj.iso8601
          when BigDecimal
            obj.to_s("F")
          else
            obj
          end
        end
      end
    end
  end
end
```

```ruby
# lib/whoosh/performance.rb
# frozen_string_literal: true

module Whoosh
  module Performance
    def self.enable_yjit!
      return unless defined?(RubyVM::YJIT)

      RubyVM::YJIT.enable unless yjit_enabled?
    end

    def self.yjit_enabled?
      defined?(RubyVM::YJIT) && RubyVM::YJIT.enabled?
    end

    def self.optimize!
      enable_yjit!
      Serialization::Json.detect_engine!
    end
  end
end
```

```ruby
def self.json(data, status: 200, headers: {})
  body = Serialization::Json.encode(data)
  if headers.empty?
    response_headers = JSON_HEADERS.dup
    response_headers["content-length"] = body.bytesize.to_s
  else
    response_headers = JSON_HEADERS.merge(headers)
```

### Task 4: Router — Static Route Cache & Frozen Empty Params

**Files:**
- Modify: `lib/whoosh/router.rb`
- Test: existing tests must still pass

- [ ] **Step 1: Read current router.rb**

- [ ] **Step 2: Add static route cache and frozen empty params**

After `freeze!`, build a flat hash for O(1) lookup of routes without params. Also return a frozen empty hash for no-param matches instead of allocating `{}` each time.

```ruby
EMPTY_PARAMS = {}.freeze

def freeze!
  @frozen = true
  build_static_cache
end

def match(method, path)
  # Try static cache first (O(1) for routes without params)
  if @static_cache
    cache_key = "#{method}:#{path}"
    cached = @static_cache[cache_key]
    return { handler: cached[:handler], params: EMPTY_PARAMS, metadata: cached[:metadata] } if cached
  end

  # Fall back to trie walk for param routes
  node = @root
  params = {}
  segments = split_path(path)

  segments.each do |segment|
    if node.children[segment]
      node = node.children[segment]
    elsif node.children[:_param]
      node = node.children[:_param]
      params[node.param_name] = segment
    else
      return nil
    end
  end

  entry = node.handlers[method]
  return nil unless entry

  { handler: entry[:handler], params: params.empty? ? EMPTY_PARAMS : params, metadata: entry[:metadata] }
end

private

def build_static_cache
  @static_cache = {}
  @routes.each do |route|
    next if route[:path].include?(":")
    match_result = @root
    split_path(route[:path]).each { |seg| match_result = match_result.children[seg] }
    entry = match_result&.handlers&.[](route[:method])
    if entry
      @static_cache["#{route[:method]}:#{route[:path]}"] = entry
    end
  end
end
```
