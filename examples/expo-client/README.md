# Expo Client for Whoosh Chat API

A React Native (Expo) client demonstrating how to consume a Whoosh API.

## Setup

```bash
cd examples/expo-client
npx expo install
npx expo start
```

## Features

- API key authentication
- Real-time chat with SSE streaming
- Health status indicator
- Works with the `examples/chat-api` backend

## Configuration

Update the API URL in `app/config.ts` to point to your Whoosh server.
